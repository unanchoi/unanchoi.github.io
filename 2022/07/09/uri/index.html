URI (Uniform Resource Identifier)

URI

  • Uniform : 리소스 식별하는 통일된 방식

  • Resource : 자원, URI로 식별할 수 있는 모든 것(제한 없음)

  • Identifier : 다른 항목과 구분하는데 필요한 정보

  • Locator, Name 또는 두 가지 모두 추가로 분류될 수 있다.

  • URL과 URN을 포괄하고 있는 개념.

  • URL(Uniform Resource Locator)

    • Locator: 리소스가 있는 위치를 지정

  • URN(Uniform Resource Name)

    • Name: 리소스에 이름을 부여

URN 이름만으로 실제 리소스를 찾을 수 있는 방법이 보편화 되지 않음.

URL 분석

  • scheme

    • 주로 프로토콜 사용(http, https, ftp ,,,)

  • userinfo
    url에 사용자 정보를 인증해야 할 때 사용
    거의 사용 안함

  • hostname

  • port

    • 일반적으로 생략, http는 80, https는 443

  • path

    • 리소스 경로, 계층적 구조

  • query

    • ?로 시작. query parameter 혹은 query string으로 불림.
    • key, value 형태

  • fragment

    • html 내부 북마크 등에 사용
